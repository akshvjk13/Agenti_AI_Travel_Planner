# AI Travel Planner Assistant

An Agentic AI-based travel planning assistant built using Python, Streamlit, Groq LLM, and modular AI tools.

## Features

- Flight recommendation
- Hotel recommendation
- Tourist place suggestions
- Weather forecasting
- Budget estimation
- AI-generated day-wise itinerary
- Streamlit user interface

## Technologies Used

- Python
- Streamlit
- Groq API
- Open-Meteo API
- JSON datasets
- Prompt Engineering

## Project Structure

```text
agents/
tools/
services/
data/
app.py
```

## How to Run

### Create Virtual Environment

```bash
python -m venv venv
```

### Activate Environment

```bash
venv\Scripts\activate
```

### Install Dependencies

```bash
pip install -r requirements.txt
```

### Run Application

```bash
streamlit run app.py
```
